Este proyecto es una aplicación de gestión de solicitudes desarrollada en Java, implementando una arquitectura limpia y guiada por pruebas (TDD). 

## Prerrequisitos

Para poder compilar y probar este proyecto en tu máquina local, necesitarás tener instalado:

* **Java JDK 17** (o superior).
* **Git** para la gestión de versiones.
* No necesitas instalar Maven, el proyecto incluye el wrapper (`mvnw`) para facilitar su ejecución.
